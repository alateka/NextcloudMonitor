# NextcloudMonitor

A new Flutter project.

## Getting Started

Create Class File on lib directory called env.dart

Use this code:

`class Env {`

  `String host = "cloud.example.es";`

  `String path = "/ocs/v2.php/apps/serverinfo/api/v1/info";`

  `int port = 4433;`

  `String token = "nextcloud_token_name";`

  `Env();`
  
`}`
